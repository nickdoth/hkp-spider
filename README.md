# hkp-spider

Crawlers for hkp.

**NOTE: This program is under active development.**

## Install & Run

Install:

```shell
git clone https://github.com/nickdoth/hkp-spider
cd hkp-spider
npm install
npm run build
```

Crawler for `/cs/detail_layer.jsp*`:

```shell
npm run cs
```

Crawler for `/tx/index.jsp*`:

```shell
npm run tx
```
