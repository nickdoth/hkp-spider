# hkp-spider

A specified crawler for hkp.

**This program is under active development**

# Install & Run

Install:

```shell
git clone https://github.com/nickdoth/hkp-spider
cd hkp-spider
npm install
npm run build
```

crawler for `/cs/detail_layer.jsp*`:

```shell
npm run cs
```

crawler for `/tx/index.jsp*`:

```shell
npm run tx
```
